## 🎯 O que buscamos?
Não queremos apenas currículos frios. Queremos saber sobre:
* **A Faísca:** O que te fez digitar o primeiro `Hello World`?
* **O "Vale da Morte":** Qual foi o erro ou desafio técnico que quase te fez desistir?
* **A Caixa de Ferramentas:** O que você estuda hoje e o que recomenda para quem está um degrau atrás de você (cursos, livros, canais).

## ✍️ Como deixar sua marca (Passo a Passo)
Contribuir é simples como um git commit:

1º Fork este repositório.

2º Copie o conteúdo do arquivo stories/template.md.

3º Crie um novo arquivo em stories/seu-nome-sobrenome.md e cole o conteúdo.

4º Escreva sua jornada! Sinta-se livre para adicionar links de cursos, livros e vídeos que mudaram seu jogo.

5º Envie um Pull Request.

## 💡 Dicas para uma boa história
Seja sincero: Histórias de superação inspiram muito mais do que histórias de perfeição absoluta.

Indique Recursos: Se aquele curso de Java ou aquele vídeo de Clean Code te ajudou, deixe o link para facilitar a vida de quem ler!

Networking: No final do seu arquivo, sinta-se à vontade para deixar seu LinkedIn ou site pessoal.

## 📜 Código de Conduta
Respeito acima de tudo. Este é um ambiente de mentoria coletiva, apoio mútuo e diversidade. Comentários ofensivos ou discriminatórios não serão aceitos nos Pull Requests.

## Por que participar?
Muitos iniciantes se sentem perdidos com o excesso de informação (a famosa paralisia por análise). Sua dica de um curso específico ou sua história de como conseguiu o primeiro emprego pode ser exatamente o norte que alguém precisa hoje.
